# random_cursor_image
generating random PNG images with randomly positioned cursors.



## dependencies

- `python>3.10`, `pillow`, `tqdm`



## usage

```bash
python3 -m random_cursor_image <IMAGE_SAVE_PATH>
python3 -m random_cursor_image -r <RANDOM_SEED> <IMAGE_SAVE_PATH>
```
